    Section 1.1 Week 1: Vectors, Matrices, and Systems

    Put a goal at the top of each lecture note. Below is just sample filler.
    🔗

    Objectives

    • Interpret vectors in \(\mathbb{R}^n\) geometrically and algebraically.
      🔗
    • Write systems of linear equations in matrix form.
      🔗
    • Use row reduction to solve linear systems.
      🔗
    🔗

    Subsection 1.1.1 Definitions

    Definition 1.1.1.

    A vector in \(\mathbb{R}^n\) is an ordered list \(\vec{v}=(v_1,v_2,\ldots,v_n)\) of real numbers.
    🔗
    🔗

    Definition 1.1.2.

    The span of vectors \(\vec{v}_1,\ldots,\vec{v}_k\) in \(\mathbb{R}^n\) is the set of all linear combinations
    \begin{equation*} \mathrm{span}\{\vec{v}_1,\ldots,\vec{v}_k\} = \{c_1\vec{v}_1+\cdots+c_k\vec{v}_k : c_1,\ldots,c_k \in \mathbb{R}\}. \end{equation*}
    🔗
    🔗
    🔗

    Subsection 1.1.2 Main Result

    Theorem 1.1.3. Row Reduction Preserves Solutions.

    If the augmented matrix of one linear system is obtained from another by elementary row operations, then the two systems have the same solution set.
    🔗
    🔗

    Proof.
    Each elementary row operation corresponds to replacing one equation by an equivalent equation or by an equation obtained from reversible algebraic operations. Therefore a vector satisfies the original system if and only if it satisfies the transformed system.
    🔗
    🔗
    🔗

    Subsection 1.1.3 Worked Example

    Example 1.1.4. A Template Row Reduction Example.

    Consider the system whose augmented matrix is
    \begin{equation*} \left[ \begin{array}{ccc|c} 1 & 2 & -1 & 3\\ 2 & 5 & 1 & 4 \end{array} \right]. \end{equation*}

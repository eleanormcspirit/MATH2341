    Section 1.1 Week 1

    Definition 1.1.1.

    A linear equation in the variables \(x_{1},x_{2},\ldots,x_{n}\) is an equation of the form
    🔗
    \begin{equation*} a_{1}x_{1}+a_{2}x_{2}+\cdots+a_{n}x_{n}=b, \end{equation*}
    where the \(a_{1},\ldots,a_{n}\) and \(b\) are constants.
    🔗
    🔗

    Example 1.1.2.

    The equation
    🔗
    \begin{equation*} 3x_{1}-2x_{2}+7x_{3}=5 \end{equation*}
    is linear.
    🔗
    🔗
    A system of linear equations is a collection of linear equations involving the same variables (e.g. \(x_{1}, x_{2}, \dots, x_{n}\) ).
    🔗

    Definition 1.1.3.

    A solution to a system is a choice of values for the variables that makes every equation in the system true.
    🔗
    The solution set is the set of all solutions to a system of linear equations.
    🔗
    🔗

    Example 1.1.4.

    The system of linear equations
    🔗
    \begin{equation*} \begin{aligned}x+y&= 5,\\ 2x-y&= 1 \end{aligned} \end{equation*}
    has solution \(x=2, y=3\) because
    🔗
    \begin{equation*} 2+3=5 \quad \textit{and}\quad 2(2)-3=1. \end{equation*}
    You can see that this is the only solution by algebraically eliminating one of the variables, or by graphing the two lines and seeing that they intersect at a single point, as shown below.
